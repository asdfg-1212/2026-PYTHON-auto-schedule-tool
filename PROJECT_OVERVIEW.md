# 📋 简化后的项目说明

## 项目定位：单日智能日程安排工具

### 核心特点
1. **一次性配置**：首次使用时设置课表和作息时间
2. **单日调度**：每天创建当天的日程
3. **智能排程**：自动安排任务到空闲时间段
4. **支持任务拆分**：长任务可拆分到多个时间段

## 🎓 课表时间规律

### 白天课程
- **上午**：8:00 开始
- **下午**：14:00 开始
- 每节课：50分钟上课 + 10分钟休息 = 60分钟计算

### 晚上课程
- **晚上**：18:30 开始
- 每节课：60分钟计算

### 示例
```
周一课表：
08:00-10:00  高等数学 (2节连上)
14:00-16:00  Python编程 (2节连上)
18:30-20:30  大学英语 (2节连上)
```

## 🔄 工作流程

### 首次使用
1. 运行程序
2. 检测到首次使用
3. 引导配置课表（每周固定）
4. 配置默认作息时间（起床、睡觉、吃饭）
5. 保存配置到 `config/user_config.json`

### 日常使用
1. 运行程序
2. 加载配置
3. 创建今天的Schedule对象
4. 从配置加载今天的课程
5. 交互式添加任务（问答形式）
6. 智能调度（自动安排到空闲时间）
7. 显示日程表
8. 保存数据

### 任务拆分说明
如果某个任务需要180分钟，但只有2个60分钟和1个90分钟的空闲段：
- 可以拆分为：60分钟 + 60分钟 + 60分钟（3个子任务）
- 或者选最大的90分钟段，提示用户剩余90分钟需要另外安排
